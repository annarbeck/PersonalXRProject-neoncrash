<h1>Neoncrash </h1> 
  <p>
    Neoncrash is a fast-paced arcade-style game set in a futuristic cyber/space environment. The player's goal is to avoid the incoming obstacles and collect points. The game becomes progressively harder, demanding quick reflexes and precise timing. 
  </p>
   
<h2> Main gameplay </h2>

<h3>Controls</h3>
    <ul>
        <li>Press the space bar to jump</li>
        <li>Release the space bar to fall</li>
        <li>Press the Down arrow key to fall faster</li>
    </ul>
    
<h3>Obstacles</h3>
    <p>Obstacles spawn from the right side of the screen and appear randomly from the top and bottom. As the game progresses, the gaps between the obstacles shrink, increasing the difficulty.</p>
    
<h3>Collision rules</h3>
    <p>Crashing into obstacles, the top, or the bottom of the screen ends the game.</p>
    
<h3>Powerups</h3>
    <p>Collecting a powerup temporarily allows the player to phase through obstacles. However, powerups do not protect againt collisions with the top or bottom of the screen.</p>
    
<h3>Scoring</h3>
    <p>The player earns 1 point for each obstacle avoided, including those passed through while a powerup is active. The game also tracks and saves the player's highscore. </p>

<h2>Features</h2>

<h3>Sound effects</h3>
<ul>
    <li>Crash and powerup activation sound effects</li>
    <li>Background music during gameplay</li>
</ul>
  
<h3>Particle effects</h3>
<ul>
    <li>Particle effect on the player while flaying</li>
    <li>Additional effect layered on top during powerup activation</li>
    <li>Explosion effect when the player crashes</li>
</ul>
  
<h3>Powerup system</h3>
<ul>
    <li>Visual and audio feedback upon activation</li>
    <li>Emission color change on the player</li>
    <li>UI slider displays remaining powerup duration</li>
</ul>
  
<h3>User Interface</h3>
<ul>
    <li>Title screen: Displays the game title and basic instructions</li>
    <li>Game Over screen: Shows a game over message and a restart button</li>
    <li>In-Game: Displays the current score and highscore, which persist across sessions</li>
</ul>
